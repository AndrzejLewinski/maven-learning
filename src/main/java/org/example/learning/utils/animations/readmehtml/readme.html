Project README

Welcome to the animation project! This README file provides an overview of the available animations and how to use them.

Animations

Here are the animations you can run:

  • runAll() - Run all predefined animations (Herbal Leaf 🌿, Four Leaf Clover 🍀, Lemon 🍋, Dots).
  • runHerbalLeaf() - Run the Herbal Leaf animation 🌿.
  • runClover() - Run the Four Leaf Clover animation 🍀.
  • runLemon() - Run the Lemon animation 🍋.
  • runDots() - Run the Dots animation.
  • greenRain() - Run the Green Rain (not conventional) animation.
  • gradientProgressBar() - Run the Gradient Progress Bar (not conventional) animation.

How to Execute

To execute a specific animation, call the corresponding method. For example:

AnimationLauncher.runHerbalLeaf();

Structure of Animations

All animation files are located under the following package structure:

org.example.learning.utils.animations

Animations Description

  • Herbal Leaf 🌿 - A calming animation featuring falling leaves.
  • Four Leaf Clover 🍀 - A lucky charm animation with four leaf clovers.
  • Lemon 🍋 - A refreshing animation with lemon-shaped items.
  • Dots - A minimalist dot animation that loops.
  • Green Rain - An unconventional animation with green-colored rain.
  • Gradient Progress Bar - A progress bar animation with a gradient effect.

Additional Information

For more details, check the code within the respective classes in org.example.learning.utils.animations.
