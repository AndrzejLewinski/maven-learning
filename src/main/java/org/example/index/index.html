1. Tablice i operacje na tablicach
🌿🍃🍀🌿🍃🍀🌿🍃🍀🌿

✅ Zadanie 1:

  • 🏙️ Zamień miejscami pierwszy i ostatni element w tablicy liczb całkowitych.
  • 🔁 Zrób to bez używania dodatkowej tablicy.
import java.util.*;

public class Main {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};
        int temp = numbers[0];
        numbers[0] = numbers[numbers.length - 1];
        numbers[numbers.length - 1] = temp;

        System.out.println(Arrays.toString(numbers));
    }
}

✅ Zadanie 2:

  • 🔍 Znajdź drugą co do wielkości liczbę w tablicy (zakładając, że tablica zawiera co najmniej dwie różne liczby).
import java.util.*;

public class Main {
    public static void main(String[] args) {
        int[] numbers = {12, 35, 1, 10, 34, 1};
        Arrays.sort(numbers);
        System.out.println("Druga największa liczba: " + numbers[numbers.length - 2]);
    }
}

✅ Zadanie 3:

  • 🔢 Zlicz liczbę wystąpień każdej liczby w tablicy i wypisz je na ekranie.
import java.util.*;

public class Main {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 2, 3, 3, 3, 4};
        Map<Integer, Integer> countMap = new HashMap<>();

        for (int num : numbers) {
            countMap.put(num, countMap.getOrDefault(num, 0) + 1);
        }

        for (Map.Entry<Integer, Integer> entry : countMap.entrySet()) {
            System.out.println(entry.getKey() + " występuje " + entry.getValue() + " razy.");
        }
    }
}

2. Rekurencja
🌿🍃🍀🌿🍃🍀🌿🍃🍀🌿

✅ Zadanie 1:

  • 🎲 Napisz rekurencyjną funkcję, która oblicza silnię liczby (n.!) dla liczby n. podanej przez użytkownika
import java.util.*;

public class Main {
    public static void main(String[] args) {
        int n = 5;
        System.out.println("Silnia z " + n + " to " + factorial(n));
    }

    public static int factorial(int n) {
        if (n == 0) return 1;
        return n * factorial(n - 1);
    }
}

✅ Zadanie 2:

  • 🧮 Stwórz funkcję rekurencyjną, która znajduje największy wspólny dzielnik (NWD) dwóch liczb.
public class Main {
    public static void main(String[] args) {
        System.out.println("NWD(56, 98) = " + gcd(56, 98));
    }

    public static int gcd(int a, int b) {
        if (b == 0) return a;
        return gcd(b, a % b);
    }
}
